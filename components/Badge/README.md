## Badge

### props

| props | type | required | default | description |
| ----- | ---- | -------- | ------- | ----------- |
| label | `String`, `undefined`, `null` | Y | `undefined` | 文案， 如果是 `undefined` 或者是 `null` ，是一个小圆点 |
| size | `Number` | N | `scale(20)` | 尺寸大小 |
| color | `String[CSS-Color]` | N | `'#FF525E'` | 颜色 |
| style | `Object` | N | `{}` | 自定义样式 |
| ----- | ---- | -------- | ------- | ----------- |
